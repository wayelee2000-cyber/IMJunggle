<TabItem value="js">

文件消息是 SDK 内置的消息类型，枚举对应 _[MessageType.FILE](../enum/web.md#message)_。

| 属性名 | 类型   | 说明                                               | 版本  |
|--------|--------|----------------------------------------------------|-------|
| name   | String | 文件名称                                           | 1.0.0 |
| url    | String | 文件地址                                           | 1.0.0 |
| size   | Number | 文件大小，用于查看或下载时显示进度条，单位为 `KB` | 1.0.0 |
| type   | String | 文件类型，常见类型包括 `word`、`excel`、`ppt`、`zip` 等 | 1.0.0 |
| extra  | String | 消息附加内容，支持 JSON 字符串，设置后不可修改     | 1.0.0 |

```js
let fileMsg = {
  name: "demo.pptx",
  url: "https://example.com/demo.pptx",
  size: 1000,
  type: "pptx",
  extra: '{"Priority":"P0"}'
}

let message = {
  conversationType: ConversationType.PRIVATE,
  conversationId: 'userId1',
  name: MessageType.FILE,
  content: fileMsg
};
```
