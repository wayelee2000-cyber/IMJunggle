<TabItem value="js">

Text messages are built-in SDK message types, corresponding to the enum _[MessageType.TEXT](../../enum/web#message)_.

| Property | Type   | Description                                         | Version |
| -------- | ------ | ------------------------------------------------- | ------- |
| content  | String | The text message content, such as user input or Emoji | 1.0.0   |
| extra    | String | Additional message content, supports JSON string; once set, it cannot be modified | 1.0.0   |

```js
let textMsg = {
  content: 'hello world',
  extra: '{"Priority":"P0"}'
}

let message = {
  conversationType: ConversationType.PRIVATE,
  conversationId: 'userId1',
  name: MessageType.TEXT,
  content: textMsg
};
```
