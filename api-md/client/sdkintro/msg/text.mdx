<TabItem value="js">

文本消息是 SDK 内置消息类型，枚举对应 _[MessageType.TEXT](../../enum/web#message)_

| 属性名  | 类型   | 说明      | 版本     |
|---------|--------|------------|----------|
| content | String | 文本消息内容，例如：用户输入的文字、Emoji 表情等 | 1.0.0    |
| extra  |  String | 消息附加内容，支持 JSON 字符串，设置后不可修改 | 1.0.0    |

```js
let textMsg = {
  content: 'hello world',
  extra: '{"Priority":"P0"}'
}

let message = {
  conversationType: ConversationType.PRIVATE,
  conversationId: 'userId1',
  name: MessageType.TEXT,
  content: textMsg
};
```
